# devsorteador

Um projeto simples em JavaScript que sorteia números aleatórios. Ideal para estudos e prática de programação front-end.

---

## 🚀 Funcionalidades

- Gerar números aleatórios dentro de um intervalo definido pelo usuário.
- Exibição do número sorteado de forma centralizada e estilizada.
- Layout responsivo para desktop e mobile.

---

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- Git/GitHub
